# 🗺️ GIS职业规划与作品集网站

> 为地图动画师岗位定制的一站式求职展示网站

## 📁 项目结构

```
.
├── career-plan/          # 职业规划网站
│   └── index.html        # 详细的岗位分析与学习规划
├── portfolio/            # 作品集网站
│   └── index.html        # 个人作品展示
└── README.md
```

## 🌐 在线访问

- **职业规划网站**: https://linqiyu-0921.github.io/gis-career-portfolio/career-plan/
- **作品集网站**: https://linqiyu-0921.github.io/gis-career-portfolio/portfolio/

## 🎯 项目简介

这是一个为GIS专业大三学生定制的求职准备项目，包含：

### 1. 职业规划网站
- 🔍 地图动画师岗位深度分析
- 📊 技能要求详解与雷达图
- 📅 一年四阶段成长计划
- 📚 精选学习资源推荐

### 2. 作品集网站
- 👤 个人介绍与技能展示
- 🎨 项目作品展示（预留作品位）
- 📑 PPT作品集入口
- 📅 学习历程时间线

## 🚀 本地预览

直接在浏览器中打开 `career-plan/index.html` 或 `portfolio/index.html` 即可预览。

## 📝 个性化修改建议

### 必须修改的内容
1. **个人头像**: 替换 `portfolio/index.html` 中的 emoji 头像为真实照片
2. **联系方式**: 更新邮箱、LinkedIn、GitHub 等链接
3. **项目内容**: 将占位项目替换为你的真实作品
4. **PPT作品集**: 创建并上传真实的 1-5 页 PPT

### 可选优化
- 添加 Google Analytics 统计访问
- 配置自定义域名
- 添加更多交互效果
